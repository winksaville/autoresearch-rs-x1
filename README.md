# autoresearch-rs-x1

An attempt to implement [autoresearch](https://github.com/karpathy/autoresearch)
in Rust.

This is the main repo of a dual-repo convention for using
a bot to help in the development of a coding project.  This main repo
contains the "what", while the partner bot repo contains "why" and "how".

The beginnings of a tool which will allow viewing the story behind the code
is [vc-x1](https://github.com/winksaville/vc-x1) maybe someday it will work.

See [jj-tips.md](jj-tips.md) on using `jj` which is the suggested version control tool
for this project.

See [notes/chores-01.md](notes/chores-01.md#use-footers-to-track-changeids-or-notes)
for some information on how git commit footers are used to track changeIDs and notes.

Until we have a better way to track the story the notes/chores-xx.md files are used to
capture plans and other information, but hopefully one this will be replaced by the
story maintained between the two repos.

In addition see [notes/todo.md](notes/todo.md) for a list of potential future
work items and [notes/done.md](notes/done.md) for a list of completed historical
work items.
